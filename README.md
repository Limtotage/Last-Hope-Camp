# 🎮 Last Hope Camp

**Last Hope Camp** is a strategy-based micromanagement game developed during a game jam with the theme **"Micromanagement"**. Your goal is simple: **build, manage, and survive**.

---

## 🚀 About the Game

In *Last Hope Camp*, you must manage workers, gather resources, and build a sustainable production system to defend your base against increasingly difficult enemy waves.

Every decision matters. Efficient resource management and smart worker placement are key to survival.

---

## 🧠 Core Gameplay

* 🛠️ Place resource nodes strategically
* 👷 Assign workers to maximize production
* ⚙️ Build a functional production chain
* ⚔️ Produce soldiers to defend your base
* 🌊 Survive incoming enemy waves

---

## 🎯 How to Play

1. **Place Resources**

   * Resources can only be placed on empty tiles (no existing resource nodes).

2. **Assign Workers**

   * Workers can be placed in **four directions** around a resource:

     * Front
     * Back
     * Left
     * Right

3. **Start Production**

   * Use structures like the **Anvil** to produce swords.

4. **Create Soldiers**

   * Click the **Soldier Card** (bottom right) to train units.
   * Each card shows required resources.

5. **Defend Your Base**

   * Enemy waves begin:

     * When the first sword is produced
     * OR when the countdown ends

---

## 💡 Developer Tip

> Place the **Anvil**, but don’t assign a worker immediately.
> First, maximize your resource production to scale faster.
